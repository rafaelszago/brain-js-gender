# Brain.js Gender

Predict gender by name. Experimental project made with Brain.js.

## Requirements

+ [Node.js](https://nodejs.org/)

## Installation or Getting Started

	git clone https://github.com/rafaelszago/brain-js-gender.git
    cd brain-js-gender
    npm install
    npm run start
